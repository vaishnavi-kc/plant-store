#  Plant Store

Plant Store is a web application for managing different types of plants such as indoor, outdoor, and flowering plants. It allows users to view and buy plant products, while the admin can manage products by adding and deleting them.

##  Technologies Used
- Frontend: Svelte
- Backend: Rust
- Database: SQLite

##  Features

### User
- View plant products
- Add to cart
- Place orders
- View order history

### Admin
- Add new plant products
- Edit and delete products
- View orders
